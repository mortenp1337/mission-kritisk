# Math Tower Defense Assets

This directory contains placeholder assets for the math tower defense game.

## Directory Structure:
- towers/ - Tower sprites (basic, rapid, area)
- enemies/ - Zombie sprites
- projectiles/ - Projectile sprites (bullet, explosion)
- ui/ - UI elements (buttons, panels, icons)
- grid/ - Grid tiles and highlights

## Asset Requirements:
All assets should be designed at appropriate sizes for 1024x768 resolution.
Grid cell size: 64x64 pixels
